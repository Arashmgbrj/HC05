# راهنمای نصب و راه‌اندازی ماژول HC-05 با آردوینو

## معرفی
ماژول بلوتوث HC-05 برای ارتباط بی‌سیم بین آردوینو و دستگاه‌های دیگر استفاده می‌شود. این راهنما مراحل کامل نصب، پیکربندی و استفاده را شرح می‌دهد.

## پیش‌نیازها
### سخت‌افزار
- برد آردوینو (Uno/Nano/Mega)
- ماژول HC-05
- مقاومت 1kΩ و 2kΩ (اختیاری - برای تقسیم ولتاژ)
- سیم‌های جامپر

### نرم‌افزار
- آردوینو IDE
- برنامه Serial Bluetooth Terminal روی موبایل

## نصب و اتصال
### اتصالات
```
HC-05   →   آردوینو
VCC     →   5V
GND     →   GND
TXD     →   RX (پین 0)
RXD     →   TX (پین 1) *با تقسیم ولتاژ
EN      →   متصل نشود
```

**تقسیم ولتاژ برای RXD:**
پین TX آردوینو → مقاومت 2kΩ → پین RXD ماژول → مقاومت 1kΩ → GND

## پیکربندی (AT Commands)
1. پین EN ماژول را به 3.3V وصل کنید
2. کد زیر را آپلود کنید:
```arduino
void setup() {
  Serial.begin(38400);
}
void loop() {
  if(Serial.available()) {
    Serial.write(Serial.read());
  }
}
```
3. در Serial Monitor دستورات زیر را وارد کنید:
```
AT                    // تست ارتباط
AT+NAME?              // مشاهده نام
AT+NAME=MY_HC05       // تغییر نام
AT+PSWD="1234"        // تغییر رمز
AT+UART=9600,0,0      // تنظیم سرعت 9600
AT+RESET              // ریست
```

## کد نمونه پایه
```arduino
void setup() {
  Serial.begin(9600);     // ارتباط با کامپیوتر
  Serial1.begin(9600);    // ارتباط با HC-05
}
void loop() {
  // دریافت از بلوتوث → ارسال به کامپیوتر
  if(Serial1.available()) {
    Serial.write(Serial1.read());
  }
  // دریافت از کامپیوتر → ارسال به بلوتوث
  if(Serial.available()) {
    Serial1.write(Serial.read());
  }
}
```

## عیب‌یابی سریع
| مشکل | راه‌حل |
|------|--------|
| ماژول روشن نمی‌شود | VCC و GND را بررسی کنید |
| ارتباط برقرار نمی‌شود | سرعت Serial را تنظیم کنید |
| داده خراب دریافت می‌شود | از تقسیم ولتاژ استفاده کنید |
| AT پاسخ نمی‌دهد | پین EN را به 3.3V وصل کنید |

## نکات مهم
1. سرعت پیش‌فرض ماژول: 9600 baud
2. رمز پیش‌فرض: 1234 یا 0000
3. محدوده کاری: تا 10 متر
4. ولتاژ منطقی: 3.3V (مراقب باشید!)

## ارتباط با موبایل
1. بلوتوث موبایل را روشن کنید
2. دستگاه "HC-05" را جستجو کنید
3. با رمز "1234" وصل شوید
4. از برنامه ترمینال برای ارتباط استفاده کنید
